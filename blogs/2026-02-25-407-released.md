<p>RubyGems 4.0.7 includes enhancements and documentation and Bundler 4.0.7 includes enhancements, bug fixes and documentation.</p>

<p>To update to the latest RubyGems you can run:</p>

<div class="language-plaintext highlighter-rouge"><div class="highlight"><pre class="highlight"><code>gem update --system [--pre]
</code></pre></div></div>

<p>To update to the latest Bundler you can run:</p>

<div class="language-plaintext highlighter-rouge"><div class="highlight"><pre class="highlight"><code>gem install bundler [--pre]
bundle update --bundler=4.0.7
</code></pre></div></div>

<h2 id="rubygems-release-notes">RubyGems Release Notes</h2>

<h3 id="enhancements">Enhancements:</h3>

<ul>
  <li>Add Gem.disable_system_update_message in setup.rb. Pull request
<a href="https://github.com/ruby/rubygems/pull/9020">#9020</a> by hyuraku</li>
  <li>Print message when signing in with an existing API key. Pull request
<a href="https://github.com/ruby/rubygems/pull/9312">#9312</a> by hsbt</li>
  <li>Installs bundler 4.0.7 as a default gem.</li>
</ul>

<h3 id="documentation">Documentation:</h3>

<ul>
  <li>Document gemspecs must be deterministic. Pull request
<a href="https://github.com/ruby/rubygems/pull/9321">#9321</a> by fxn</li>
  <li>Remove “##” from a comment to require. Pull request
<a href="https://github.com/ruby/rubygems/pull/9306">#9306</a> by tompng</li>
</ul>

<h2 id="bundler-release-notes">Bundler Release Notes</h2>

<h3 id="enhancements-1">Enhancements:</h3>

<ul>
  <li>Don’t check whether a plugin needs to be installed: <a href="https://github.com/ruby/rubygems/pull/9328">#9328</a></li>
  <li>[rust gem] Major improvements for gem scaffolding (rebased) <a href="https://github.com/ruby/rubygems/pull/8455">#8455</a></li>
  <li>Fix(bundler): only preload git sources for requested groups <a href="https://github.com/ruby/rubygems/pull/9234">#9234</a></li>
  <li>Raise error when gem contains capital letters <a href="https://github.com/ruby/rubygems/pull/5432">#5432</a></li>
</ul>

<h3 id="bug-fixes">Bug fixes:</h3>

<ul>
  <li>Fix Bundler crashing when it tries to install plugin: <a href="https://github.com/ruby/rubygems/pull/9335">#9335</a></li>
  <li>Run git operations in parallel (take 2): <a href="https://github.com/ruby/rubygems/pull/9323">#9323</a></li>
  <li>Add support for help flag in plugin commands <a href="https://github.com/ruby/rubygems/pull/9263">#9263</a></li>
</ul>

<h3 id="documentation-1">Documentation:</h3>

<ul>
  <li>[DOC] Fix link in Bundler <a href="https://github.com/ruby/rubygems/pull/9315">#9315</a></li>
</ul>
