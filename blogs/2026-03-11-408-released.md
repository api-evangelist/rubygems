<p>RubyGems 4.0.8 includes enhancements and documentation and Bundler 4.0.8 includes enhancements and bug fixes.</p>

<p>To update to the latest RubyGems you can run:</p>

<div class="language-plaintext highlighter-rouge"><div class="highlight"><pre class="highlight"><code>gem update --system [--pre]
</code></pre></div></div>

<p>To update to the latest Bundler you can run:</p>

<div class="language-plaintext highlighter-rouge"><div class="highlight"><pre class="highlight"><code>gem install bundler [--pre]
bundle update --bundler=4.0.8
</code></pre></div></div>

<h2 id="rubygems-release-notes">RubyGems Release Notes</h2>

<h3 id="enhancements">Enhancements:</h3>

<ul>
  <li>Use JSON for cargo metadata parsing. Pull request
<a href="https://github.com/ruby/rubygems/pull/9373">#9373</a> by hsbt</li>
  <li>Fix NameError in Gem::Request.get_proxy_from_env when requiring
rubygems/request directly. Pull request
<a href="https://github.com/ruby/rubygems/pull/9362">#9362</a> by afurm</li>
  <li>Installs bundler 4.0.8 as a default gem.</li>
</ul>

<h3 id="documentation">Documentation:</h3>

<ul>
  <li>Unify Compact Index API naming. Pull request
<a href="https://github.com/ruby/rubygems/pull/9372">#9372</a> by simi</li>
</ul>

<h2 id="bundler-release-notes">Bundler Release Notes</h2>

<h3 id="enhancements-1">Enhancements:</h3>

<ul>
  <li>Add a new Bundler config to control how many specs are fetched <a href="https://github.com/ruby/rubygems/pull/9363">#9363</a></li>
  <li>Restrict GitHub Actions workflow permissions for newgem <a href="https://github.com/ruby/rubygems/pull/9361">#9361</a></li>
</ul>

<h3 id="bug-fixes">Bug fixes:</h3>

<ul>
  <li>Fix plugin new version not registering <a href="https://github.com/ruby/rubygems/pull/9355">#9355</a></li>
</ul>
