RubyGems 4.0.9 includes enhancements, bug fixes and documentation and Bundler 4.0.9 includes enhancements and bug fixes. To update to the latest RubyGems you can run: gem update --system [--pre] To update to the latest Bundler you can run: gem install bundler [--pre] bundle update --bundler=4.0.9 RubyGems Release Notes Enhancements: Fix: include owner role in gem owner . Pull request #9403 by gjtorikian Installs bundler 4.0.9 as a default gem.
