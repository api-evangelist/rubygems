<p>RubyGems 4.0.9 includes enhancements, bug fixes and documentation and Bundler 4.0.9 includes enhancements and bug fixes.</p>

<p>To update to the latest RubyGems you can run:</p>

<div class="language-plaintext highlighter-rouge"><div class="highlight"><pre class="highlight"><code>gem update --system [--pre]
</code></pre></div></div>

<p>To update to the latest Bundler you can run:</p>

<div class="language-plaintext highlighter-rouge"><div class="highlight"><pre class="highlight"><code>gem install bundler [--pre]
bundle update --bundler=4.0.9
</code></pre></div></div>

<h2 id="rubygems-release-notes">RubyGems Release Notes</h2>

<h3 id="enhancements">Enhancements:</h3>

<ul>
  <li>Fix: include owner role in <code class="language-plaintext highlighter-rouge">gem owner</code>. Pull request <a href="https://github.com/ruby/rubygems/pull/9403">#9403</a> by gjtorikian</li>
  <li>Installs bundler 4.0.9 as a default gem.</li>
</ul>

<h3 id="bug-fixes">Bug fixes:</h3>

<ul>
  <li>Fix: Ensure trailing slash is added to source URIs added via gem sources. Pull request <a href="https://github.com/ruby/rubygems/pull/9055">#9055</a> by zirni</li>
</ul>

<h3 id="documentation">Documentation:</h3>

<ul>
  <li>[DOC] Fix link. Pull request <a href="https://github.com/ruby/rubygems/pull/9409">#9409</a> by BurdetteLamar</li>
</ul>

<h2 id="bundler-release-notes">Bundler Release Notes</h2>

<h3 id="enhancements-1">Enhancements:</h3>

<ul>
  <li>Check the git version only <strong>once</strong> per <code class="language-plaintext highlighter-rouge">bundle install</code>. Pull request <a href="https://github.com/ruby/rubygems/pull/9406">#9406</a> by Edouard-chin</li>
  <li>Normalize the number of workers when performing parallel operations. Pull request <a href="https://github.com/ruby/rubygems/pull/9400">#9400</a> by Edouard-chin</li>
  <li>Add exponential backoff to bundler retries. Pull request <a href="https://github.com/ruby/rubygems/pull/9163">#9163</a> by ChrisBr</li>
  <li>Introduce a priority queue. Pull request <a href="https://github.com/ruby/rubygems/pull/9389">#9389</a> by Edouard-chin</li>
  <li>Split the download and install process of a gem. Pull request <a href="https://github.com/ruby/rubygems/pull/9381">#9381</a> by Edouard-chin</li>
</ul>

<h3 id="bug-fixes-1">Bug fixes:</h3>

<ul>
  <li>Retry git fetch without –depth for dumb HTTP transport. Pull request <a href="https://github.com/ruby/rubygems/pull/9405">#9405</a> by hsbt</li>
</ul>
