<p>RubyGems 4.0.10 includes enhancements and bug fixes and Bundler 4.0.10 includes enhancements and bug fixes.</p>

<p>To update to the latest RubyGems you can run:</p>

<div class="language-plaintext highlighter-rouge"><div class="highlight"><pre class="highlight"><code>gem update --system [--pre]
</code></pre></div></div>

<p>To update to the latest Bundler you can run:</p>

<div class="language-plaintext highlighter-rouge"><div class="highlight"><pre class="highlight"><code>gem install bundler [--pre]
bundle update --bundler=4.0.10
</code></pre></div></div>

<h2 id="rubygems-release-notes">RubyGems Release Notes</h2>

<h3 id="enhancements">Enhancements:</h3>

<ul>
  <li>Ignore warnings with spec different platforms. Pull request <a href="https://github.com/ruby/rubygems/pull/8508">#8508</a> by hsbt</li>
  <li>Better algorithm for sorting gem version. Pull request <a href="https://github.com/ruby/rubygems/pull/9421">#9421</a> by Edouard-chin</li>
  <li>Update SPDX license list as of 2026-02-20. Pull request <a href="https://github.com/ruby/rubygems/pull/9434">#9434</a> by hsbt</li>
  <li>Installs bundler 4.0.10 as a default gem.</li>
</ul>

<h3 id="bug-fixes">Bug fixes:</h3>

<ul>
  <li>Register native extension files in default spec map. Pull request <a href="https://github.com/ruby/rubygems/pull/9431">#9431</a> by hsbt</li>
  <li>Fix NoMethodError in Gem.try_activate when activation conflicts occur. Pull request <a href="https://github.com/ruby/rubygems/pull/9404">#9404</a> by hsbt</li>
</ul>

<h2 id="bundler-release-notes">Bundler Release Notes</h2>

<h3 id="enhancements-1">Enhancements:</h3>

<ul>
  <li>Ignore warnings with spec different platforms. Pull request <a href="https://github.com/ruby/rubygems/pull/8508">#8508</a> by hsbt</li>
  <li>Improve error message when current platform is not in lockfile. Pull request <a href="https://github.com/ruby/rubygems/pull/9439">#9439</a> by 55728</li>
  <li>Cache package version selection. Pull request <a href="https://github.com/ruby/rubygems/pull/9410">#9410</a> by Edouard-chin</li>
  <li>Check happy path first when comparing gem version. Pull request <a href="https://github.com/ruby/rubygems/pull/9417">#9417</a> by Edouard-chin</li>
  <li>[feature] default_cli_command for config what command bundler runs when no specific command is provided. Pull request <a href="https://github.com/ruby/rubygems/pull/8886">#8886</a> by jonbarlo</li>
  <li>Introduce a fast path for comparing Gem::Version. Pull request <a href="https://github.com/ruby/rubygems/pull/9414">#9414</a> by Edouard-chin</li>
</ul>

<h3 id="bug-fixes-1">Bug fixes:</h3>

<ul>
  <li>Restore rb_sys dependency for Rust. Pull request <a href="https://github.com/ruby/rubygems/pull/9416">#9416</a> by bangseongbeom</li>
</ul>
