<p>RubyGems 4.0.11 includes enhancements and Bundler 4.0.11 includes enhancements, bug fixes and documentation.</p>

<p>To update to the latest RubyGems you can run:</p>

<div class="language-plaintext highlighter-rouge"><div class="highlight"><pre class="highlight"><code>gem update --system [--pre]
</code></pre></div></div>

<p>To update to the latest Bundler you can run:</p>

<div class="language-plaintext highlighter-rouge"><div class="highlight"><pre class="highlight"><code>gem install bundler [--pre]
bundle update --bundler=4.0.11
</code></pre></div></div>

<h2 id="rubygems-release-notes">RubyGems Release Notes</h2>

<h3 id="enhancements">Enhancements:</h3>

<ul>
  <li>Add commented-out rubygems_mfa_required to bundle gem template. Pull request <a href="https://github.com/ruby/rubygems/pull/9487">#9487</a> by MatheusRich</li>
  <li>Clarify the name and meaning of the first argument to <code class="language-plaintext highlighter-rouge">gem spec</code>. Pull request <a href="https://github.com/ruby/rubygems/pull/9476">#9476</a> by eregon</li>
  <li>Installs bundler 4.0.11 as a default gem.</li>
</ul>

<h2 id="bundler-release-notes">Bundler Release Notes</h2>

<h3 id="enhancements-1">Enhancements:</h3>

<ul>
  <li>Update gem creation guide URL to rubygems.org. Pull request <a href="https://github.com/ruby/rubygems/pull/9500">#9500</a> by nissyi-gh</li>
  <li>Lock the checksum of Bundler itself in the lockfile. Pull request <a href="https://github.com/ruby/rubygems/pull/9366">#9366</a> by Edouard-chin</li>
</ul>

<h3 id="bug-fixes">Bug fixes:</h3>

<ul>
  <li>Fix installing gems with native extensions + transitive dependencies. Pull request <a href="https://github.com/ruby/rubygems/pull/9477">#9477</a> by nicholasdower</li>
  <li>Fix the bundler version not being updated in dev/test lockfile. Pull request <a href="https://github.com/ruby/rubygems/pull/9463">#9463</a> by Edouard-chin</li>
  <li>Ensure the release CI doesn’t break due to the Bundler checksum feature. Pull request <a href="https://github.com/ruby/rubygems/pull/9436">#9436</a> by Edouard-chin</li>
</ul>

<h3 id="documentation">Documentation:</h3>

<ul>
  <li>Fix formatting for BUNDLE_PREFER_PATCH variable in man page. Pull request <a href="https://github.com/ruby/rubygems/pull/9474">#9474</a> by toy</li>
</ul>
