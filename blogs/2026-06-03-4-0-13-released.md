RubyGems 4.0.13 and Bundler 4.0.13 have been released with multiple security and functionality improvements. The updates include preventing extraction escapes via symlinks, closing stdin in popen2e, and adding a cooldown feature to delay newly published gems for community vetting. Users can update via gem update --system for RubyGems or gem install bundler for Bundler.
